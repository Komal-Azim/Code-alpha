# Code-alpha
Network Traffic Packet Capture and Analysis using Python & Scapy 
Project
I successfully captured and analyzed live network traffic packets using Python. This project helped me understand how data flows across a network and how protocols such as TCP work at the packet level.
Captured packet details included:
🔹 Source IP
🔹 Destination IP
🔹 Protocol
🔹 Packet Length
🔹 Source & Destination Ports
🔹 TCP Flags
🔹 Payload status
This packet capture was performed in a controlled lab environment for learning purposes only.
Through this task, I improved my understanding of computer networks, packet structure, and cybersecurity fundamentals.
